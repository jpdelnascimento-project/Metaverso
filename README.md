# Projeto Final: Atividade Metaverso
## Curso de Web 3.0 - Residência em TIC

### 📋 Conceito do Projeto
Uma sala de estar minimalista desenvolvida no Unity contendo os seguintes elementos 3D:
* **Sofá** (configurada e otimizada como *Prefab*)
* **Poltrona** (configurada e otimizada como *Prefab*)
* **Mesa**
* **Aparelho de TV**
* **Vaso de Planta** (cilindro decorativo)

---

### 🕹️ Controles para PC
* **Teclas WASD / Setas do teclado:** Utilizadas para navegar livremente pelo ambiente.
* **Movimento do Mouse:** Controla a visão, permitindo girar a câmera e olhar ao redor de forma fluida.

---

### 🛠️ Configurações Técnicas Implementadas
* **Plataforma VR:** Projeto alterado e ativo para a plataforma **Android (Meta Quest)** nas *Build Settings*.
* **Organização:** Hierarquia limpa e organizada utilizando objetos lógicos vazios (`_CENARIO` e `_OBJETOS_INTERATIVOS`) para uma nomenclatura clara.
* **Ambientação:** Configuração de um **Skybox procedural** para a iluminação e renderização natural do céu.
